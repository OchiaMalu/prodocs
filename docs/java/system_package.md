:::tip `Applet` 技术较为古老，已经不再使用了

`Applet` 是 Java 在网页上嵌套的程序，是采用绘图的方式完成的显示；而 `Application`
是在主方法中运行，通过命令行执行。随着时间的发展，`Applet` 程序已经不再使用了，如果要再实现同样的功能，会利用 HTML5
中的 `Canvas` 技术实现。

:::
